## ⚙️ Setup & Installation

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/Namit-07/Web-Dev.git
cd "Sigma Web Dev course/Projects"
```

2️⃣ Install Dependencies (for React/Node projects)
```bash
Copy code
npm install
```

3️⃣ Run the Project
```bash
Copy code
npm start
```

Copy code
```bash

```
